# Variable & Geometry Enforcement Rules

> **Foundation Source:** `https://ui.shadcn.com/create?preset=b2H8qFFOtM`\
> **Preset ID:** `b2HRtJbF9k`\
> **Aesthetic Directive:** Avant-Garde Cyan/Mist — Intentional Minimalism\
> **Version:** 2.0 (Zero-Tolerance Enforcement)

---

## Purpose

To maintain the integrity of the stylized, non-realistic **Cyan/Mist** aesthetic
generated by the `b2HRtJbF9k` preset, all styling must strictly utilize the CSS
variables generated by the shadcn CLI. Additionally, all geometry must remain
strictly rectilinear as an architectural constraint of the Avant-Garde design
language.

Any agent generation attempt failing these rules must be **automatically
rejected** and **self-corrected** to use the defined variable taxonomy and sharp
geometry.

---

## Enforcement Directives

### 1. Strict Preset Compliance

You **MUST ONLY** use the variables provided by the `b2HRtJbF9k` preset in
`src/index.css`.

- **FORBIDDEN:** Manual creation of CSS variables (e.g., `--beskar-base`,
  `--forge-glow`, `--custom-radius`, `--my-color`).
- **FORBIDDEN:** Extending the theme with custom CSS variables in any file.
- **RULE:** If a variable does not exist in the generated preset, you do not use
  it. You do not create it. You map your need to the closest existing semantic
  variable.

### 2. Banned Hardcoding

Raw hex codes, RGB, RGBA, HSL, or OKLCH values **CANNOT** be used under any
circumstance in:

- Inline React styles (`style={{ color: '#000' }}`,
  `style={{ backgroundColor: 'rgb(0,0,0)' }}`)
- JSX logic or conditional rendering
  (`className={isActive ? 'text-[#fff]' : 'text-[#000]'}`)
- Tailwind arbitrary square-bracket values for color (`bg-[#F59E0B]`,
  `text-[rgb(0,0,0)]`, `border-[hsl(200,50%,50%)]`)
- CSS-in-JS objects (`css={{ color: '#fff' }}`)
- `<style>` blocks or global CSS overrides
- JavaScript/TypeScript color constants (`const COLOR = '#0ff'`)

**Zero tolerance.** There are no exceptions for "special cases," "accents,"
"highlights," or "one-off overrides."

### 3. Banned Curved Geometry (No-Curve Mandate)

Curved lines, rounded corners, and circular shapes are **STRICTLY FORBIDDEN**.
This is a non-negotiable architectural constraint of the Avant-Garde aesthetic.

- **NEVER USE:**
  - `rounded`
  - `rounded-sm`
  - `rounded-md`
  - `rounded-lg`
  - `rounded-xl`
  - `rounded-2xl`
  - `rounded-3xl`
  - `rounded-full`
  - `border-radius` (in any CSS)
  - `circle` (shape)
  - `ellipse` (shape)
  - `radial-gradient`
  - Tailwind arbitrary bracket rounding (`rounded-[4px]`, `rounded-[50%]`)

- **MANDATORY:**
  - If a base Shadcn primitive imports with default rounding, you **MUST**
    override it explicitly with `rounded-none`.
  - All corners must remain at `0px` (sharp 90-degree angles).
  - All geometry must be rectilinear: rectangles, squares, straight lines, sharp
    angles only.

- **NO EXCEPTIONS** for:
  - Decorative elements
  - Icons
  - Avatars
  - Badges
  - Buttons
  - Cards
  - Modals
  - Inputs
  - Any other UI element

### 4. Allowed Usage

The following are the **only** permitted styling tools:

#### Layout & Spacing

- Standard Tailwind layout utilities: `flex`, `grid`, `block`, `inline`,
  `hidden`, `p-4`, `m-2`, `gap-2`, `w-full`, `h-screen`, `min-h-0`,
  `overflow-auto`, etc.
- Standard Tailwind sizing utilities: `w-`, `h-`, `min-w-`, `max-h-`, etc.
- Standard Tailwind typography utilities: `text-sm`, `font-bold`,
  `leading-tight`, `tracking-wide`, etc.
- Standard Tailwind effect utilities: `shadow-sm`, `shadow-md`, `opacity-50`,
  `blur-sm`, etc.
- Standard Tailwind transition utilities: `transition`, `duration-200`,
  `ease-in-out`, etc.

#### Semantic Color Variables (from `b2HRtJbF9k` preset)

- **Global:** `bg-background`, `text-foreground`, `border-border`, `ring-ring`
- **Primary:** `bg-primary`, `text-primary`, `text-primary-foreground`
- **Secondary:** `bg-secondary`, `text-secondary`, `text-secondary-foreground`
- **Muted:** `bg-muted`, `text-muted`, `text-muted-foreground`
- **Accent:** `bg-accent`, `text-accent`, `text-accent-foreground`
- **Destructive:** `bg-destructive`, `text-destructive`,
  `text-destructive-foreground`
- **Card:** `bg-card`, `text-card-foreground`
- **Popover:** `bg-popover`, `text-popover-foreground`
- **Opacity modifiers on allowed variables only:** `bg-primary/90`,
  `text-muted/50`, `border-border/20`

#### Geometry

- `rounded-none` (mandatory override for primitives with default rounding)
- `border` (when mapped to `border-border`)
- `border-t`, `border-b`, `border-l`, `border-r` (when mapped to
  `border-border`)

## Self-Correction Protocol

Any generation attempt failing these rules **MUST** be automatically rejected by
the agent. The agent **MUST** self-correct immediately by executing the
following steps in order:

1. **Strip Hardcoded Colors:** Remove all hex codes, RGB, RGBA, HSL, and OKLCH
   values from classes, inline styles, JSX logic, and constants.
2. **Strip Curved Geometry:** Remove all `rounded-*` classes, `border-radius`
   properties, `circle` shapes, and `radial-gradient` values. Replace with
   `rounded-none` where geometry is required.
3. **Re-map to Taxonomy:** Replace all removed color values with the closest
   semantic variable from the `b2HRtJbF9k` preset.
4. **Re-audit Geometry:** Verify every element in the file has sharp corners.
   Override any remaining primitive defaults with `rounded-none`.
5. **Final Scan:** Run a regex search across the file for banned patterns:
   - `#` followed by hex characters
   - `rgb(`, `rgba(`, `hsl(`, `oklch(`
   - `rounded-` (excluding `rounded-none`)
   - `border-radius`
   - `[` and `]` containing color values
6. **Do Not Deliver** the code to the user until all checks pass.

## Violation Examples (What NOT to Do)

```jsx
// ❌ HARDCODED COLOR — FORBIDDEN
<div className="bg-[#0ff] text-[#000]">

// ❌ INLINE STYLE COLOR — FORBIDDEN
<div style={{ backgroundColor: 'rgb(0, 255, 255)' }}>

// ❌ ARBITRARY ROUNDING — FORBIDDEN
<button className="rounded-[4px]">

// ❌ ROUNDED CORNERS — FORBIDDEN
<div className="rounded-lg rounded-md rounded-sm">

// ❌ CUSTOM CSS VARIABLE — FORBIDDEN
<style>{`:root { --forge-glow: #0ff; }`}</style>

// ❌ TAILWIND ARBITRARY COLOR — FORBIDDEN
<span className="text-[hsl(200,50%,50%)]">

// ❌ CIRCULAR SHAPE — FORBIDDEN
<div className="rounded-full w-10 h-10">
```

```jsx
// ✅ CORRECT — Semantic Variables + Sharp Geometry
<div className="bg-background text-foreground border border-border rounded-none">
   <button className="bg-primary text-primary-foreground rounded-none px-4 py-2">
      Action
   </button>
</div>;
```
